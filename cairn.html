NAME
     DM-07 CAIRN -- Dual-Mode Quantizer / Voltage Controlled Oscillator

STATUS
     Active: yes
     Revision: 02
     First logged: 2026.09.14

DESCRIPTION
     CAIRN accepts an unquantized CV signal at its primary input and
     constrains it to user-selectable scale intervals. In VCO mode,
     the same pitch-processing architecture generates its own signal,
     producing a clean chromatic output across five octaves with
     sub-cent accuracy.

     The dual-mode architecture allows a single 10HP module to serve
     as either a precision quantizer or a tracking oscillator
     depending on system requirements.

SPECIFICATIONS
     Width ..............  10HP
     Depth ..............  25mm
     Current ............  +12V: 45mA / -12V: 8mA
     ADC ................  ADS8866, 16-bit, 100kSPS
     DAC ................  DAC8551, 16-bit
     MCU ................  RP2040 / RP2350
     Pitch accuracy .....  +/-0.5 cent / 5 octaves
     Modes ..............  QUANTIZER / VCO
